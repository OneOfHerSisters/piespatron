# Инструкция для создания favicon.png

Для создания favicon.png из SVG файла:

1. Откройте favicon.svg в графическом редакторе (GIMP, Photoshop, или онлайн конвертер)
2. Экспортируйте как PNG с размерами:
   - 16x16 пикселей (основной favicon)
   - 32x32 пикселей (для высокого разрешения)
   - 48x48 пикселей (для Windows)

3. Сохраните как favicon.png в папке assets/img/

Альтернативно, используйте онлайн конвертер SVG в PNG:
- https://convertio.co/svg-png/
- https://cloudconvert.com/svg-to-png

Рекомендуемые размеры для favicon:
- 16x16px (стандартный)
- 32x32px (высокое разрешение)
- 48x48px (Windows)
- 180x180px (Apple Touch Icon)
