Co już zrobiłam

2025-11-13 (Dzień 1)
- Strona jest na GitHub Pages
- Dodałam podstawowe SEO:
  - Meta tagi (tytuł, opis)
  - Open Graph (żeby ładnie wyglądało na Facebooku)
  - Google Search Console verification (już zweryfikowane)
  - robots.txt i sitemap.xml
  - Poprawne HTML (h1, alt dla obrazków)
- Google Analytics - kod jest, ale trzeba jeszcze dodać swój ID

Co jeszcze muszę zrobić:
1. Skonfigurować Google Analytics (dostać ID)
2. Zgłosić stronę do Google Search Console (dodać sitemap)
3. Zgłosić do Bing Webmaster Tools
4. DuckDuckGo powinno samo znaleźć (mam robots.txt)
